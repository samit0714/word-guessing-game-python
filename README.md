This is a simple and engaging Word Guessing Game where players try to guess a hidden word one letter at a time. The game challenges your vocabulary and problem-solving skills while providing an interactive experience.

## Features
- Interactive command-line interface
- Random word selection (can be customized)
- Tracks correct and incorrect guesses
- Displays progress after each guess
- Option to replay the game

## How to Play
1. Run the game script.
2. Guess one letter at a time.
3. Reveal the hidden word before running out of attempts.

## Tech Stack
- Language: Python
- No external dependencies required

Perfect for beginners learning programming concepts like loops, conditionals, and arrays/strings.
